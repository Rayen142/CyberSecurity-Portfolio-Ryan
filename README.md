# Cyber Security Portfolio - Ryan Hanif Dwihandoyo

Repository ini berisi dokumentasi lengkap Security Assessment (Penetration Testing) yang dilakukan terhadap target simulasi `dibimbing-shop`.

## Ringkasan Proyek
* **Target:** Web Application (E-Commerce)
* **Tools:** Nmap, Burp Suite, SQLMap, Gobuster, Nikto, Whoami
* **Temuan Utama:**
    1.  Critical: Price Tampering (Business Logic Flaw)
    2.  Critical: SQL Injection
    3.  Critical: Unrestricted File Upload (RCE)
